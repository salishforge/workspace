### 2. Call Agents
```bash

# Get Flint's status
curl -X POST https://100.97.161.7:3000/api/rpc/call \
  -H "Authorization: Bearer $TOKEN" \
  -H 'Content-Type: application/json' \
  -d '{
    "sourceAgent": "john-broker",
    "targetAgent": "flint",
    "capability": "status",
    "params": {},
    "timeout": 10000
```
